# 🎤🔁 Real-Time Multilingual Voice Translator

A web-based real-time voice translation system built using HTML, CSS, and JavaScript.  
This application captures speech input in one language, translates it into a user-selected target language, and provides audible output in the translated language.

---

## 🌍 Features

- 🎤 Speech Recognition (Voice to Text)
- 🌐 Automatic Language Detection
- 🔁 Real-Time Text Translation
- 🔊 Audio Output in Selected Language
- 🌎 Supports Multiple Languages:
  - English
  - Hindi
  - Telugu
  - Tamil
  - Kannada
  - Malayalam
  - Chinese
  - Spanish
  - French
  - German

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Web Speech API (SpeechRecognition & SpeechSynthesis)
- Google Translate Free API Endpoint
